Web存储

Author: Cedrus-P

存储

cookie

  • 本身用于浏览器和server通讯

  • 被借用到本地存储

  • 可以用document.cookie = '...'来修改

  • 相同的key会覆盖，不同的key会追加

缺点
  • 存储大小只有4kb
  • http请求时需要发送到服务端，增加请求数量
  • 只能用document.cookie = '...'来修改

localStorage sessionStorage 区别

  • HTML5专门为存储设计，最大可以存5M

  • API简单易用 setItem getItem

  • 不会随着http请求发送出去

    区别

    • localStorage 数据会永久存储，除非代码或手动删除
    • sessionStorage 数据只会存储在当前会话，浏览器关闭后自动清空
    • 一般用 localStorage 多一些
